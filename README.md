# 스터디 개요

- 개발자 취업을 위한 RDBMS 지식을 채우는 것을 목표로 면접시 자주 등장하는 주제를 속성으로 함께 공부하고 토론합니다.
- Real MySQL 8.0 1권의 4, 5, 8, 9, 10 장을 공부하며, 주어진 기간에 주어진 챕터를 정리하고 GitHub 레포에 PR을 보내는 방식으로 진행합니다.
- 주 2회마다 스프린트를 가지며 스프린트마다 해당 챕터에 대해 본인이 이해한 내용을 정리하여 PR을 마감합니다. (총 5개의 챕터 2.5주 요일 미정)
- PR이 마감된 다음 2일간 리뷰 데이로 다른 스터디원의 PR을 리뷰하며 이해가 가지 않는 부분이나, 추가됐으면 좋을만한 내용을 리뷰합니다.

# PR, 코드리뷰

스터디 레포를 fork하셔서 본인 이름으로 된 폴더 안에 본인이 마크다운 형식으로 정리한 내용을 PR보내시면 됩니다.

PR 제목 컨벤션

```
[X주차] 홍길동 - 4장 정리
```

PR은 리뷰데이 전날 23:59 까지 마감시간이며, 지각시 벌금이 있습니다.

리뷰엔 별도의 형식이 없습니다. 자유롭게 해주시면 됩니다.

리뷰데이는 PR 마감 다음날부터 2일차 23:59 까지 마감이며, 마찬가지로 구성원들의 리뷰를 마치지 않았을 시 벌금이 존재합니다.

PR이나 리뷰 지각시 다음 스프린트 전까지 마감하도록 합니다. 지각한 PR에 대해선 리뷰는 자유입니다.
